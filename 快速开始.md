# Frontend Deploy Manager - 快速开始指南

## 🎉 项目已完成！

恭喜！**Frontend Deploy Manager** 已经完成所有核心功能开发！

## 🚀 快速启动

### 方式一：使用启动脚本（推荐）
```bash
# Windows
start.bat

# 或直接使用 npm
npm run dev
```

### 方式二：手动启动
```bash
# 1. 安装依赖（如果还没安装）
npm install

# 2. 启动开发服务器
npm run dev
```

## 📋 功能清单

✅ 项目管理 - 增删改查、分组、Git 状态  
✅ 凭证管理 - 服务器/SVN 凭证、加密存储  
✅ SVN 发布 - 构建 + SVN 提交  
✅ 服务器发布 - SFTP 上传、进度显示  
✅ 混合发布 - 单次构建、多目标发布  
✅ 配置导入导出 - JSON 备份  
✅ 数据库 - SQLite 完整设计  
✅ 加密 - AES-256 密码加密  
✅ UI - Element Plus 现代界面

## ⚠️ 常见问题

### 1. Electron 安装失败
```bash
npm uninstall electron
npm install electron@28.2.3
```

### 2. better-sqlite3 编译失败
项目已使用 sql.js 替代 better-sqlite3，无需编译。

### 3. 端口被占用
Vite 默认使用 5173 端口，如被占用会自动尝试下一个端口。
