# Frontend Deploy Manager

一款基于 Electron + Vue3 + TypeScript 的桌面应用，用于管理前端项目的构建和部署流程。

## 🎉 功能特点

- ✅ **项目管理**: 项目增删改查、分组管理
- ✅ **凭证管理**: 服务器和 SVN 凭证的加密存储
- ✅ **SVN 发布**: 构建命令执行 → SVN 握交
- ✅ **服务器发布**: SFTP 上传构建产物,- ✅ **混合发布**: 单次构建,多次发布
- ✅ **配置导入导出**: JSON 格式配置备份
- ✅ **数据库**: SQLite3 本地存储
- ✅ **加密**: AES-256 加密所有敏感信息
- ✅ **日志**: 完整的操作日志记录
- ✅ **错误处理**: 友好的错误提示

- ✅ **UI**: 现代化的 Element Plus 界面

- ✅ **路由**: Vue Router 完整路由系统
- ✅ **状态管理**: Pinia 状态管理
- ✅ **Type安全**: 完整的 TypeScript 类型定义

- ✅ **模块化**: 清晰的项目结构

- ✅ **文档**: 详细的技术方案

- ✅ **开发规范**: 遵循最佳实践
- ✅ **生产就绪**: 可直接构建和部署
