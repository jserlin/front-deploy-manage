# Frontend Deploy Manager - 安装和运行指南

## ⚠️ 重要提示

如果自动安装遇到问题，请按照以下步骤手动安装：

## 📦 手动安装步骤

### 1. 清理旧的依赖（如果有）
```bash
rm -rf node_modules package-lock.json
```

### 2. 安装依赖
```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install

# 或使用 yarn
yarn install
```

### 3. 启动开发服务器
```bash
npm run dev
```

## 🚀 快速启动（如果依赖已安装）

如果依赖已经安装完成，直接运行：
```bash
npm run dev
```

## 📦 访问应用

启动成功后，应用将自动打开窗口。
- 地址: http://localhost:5173

## ❓ 常见问题

### 问题 1: Electron 安装失败
```bash
# 解决方案：
npm uninstall electron
npm install electron@28.2.3
```

### 问题 2: better-sqlite3 编译失败
项目已改用 sql.js（纯 JavaScript 实现），无需编译。

### 问题 3: 端口被占用
```bash
# 检查端口
netstat -ano | findstr "5173"

# 如果被占用，修改 vite.config.ts 中的端口
```

### 问题 4: 依赖冲突
```bash
# 强制重新安装
npm install --force
```
